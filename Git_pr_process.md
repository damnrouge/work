| Step                  | Purpose (Developer View)                                | Command Line (Git)                                                               | VS Code UI                                                  |
| --------------------- | ------------------------------------------------------- | -------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **1. Sync main**      | Start from latest stable code to avoid conflicts        | `bash\n git checkout main\n git pull origin main\n`                              | Bottom-left branch → select `main` → Click **Sync Changes** |
| **2. Create branch**  | Isolate your feature/bug work safely                    | `bash\n git checkout -b feature/login-validation\n`                              | Click branch name → **Create new branch** → enter name      |
| **3. Make changes**   | Modify code in working directory                        | *(edit files)*                                                                   | Open files → edit → save                                    |
| **4. Stage changes**  | Select what goes into next commit (controlled snapshot) | `bash\n git add .\n` or `bash\n git add file.js\n`                               | Source Control → click `+` (stage files)                    |
| **5. Commit**         | Create version checkpoint with message                  | `bash\n git commit -m "Add login validation"\n`                                  | Enter message → click ✔️ Commit                             |
| **6. Push branch**    | Upload branch to GitHub and link upstream               | `bash\n git push -u origin feature/login-validation\n`                           | Click **Publish Branch**                                    |
| **7. Create PR**      | Request merge + trigger review/CI checks                | *(done in GitHub UI)*                                                            | Click **Create Pull Request** popup                         |
| **8. Address review** | Fix feedback → update same PR                           | `bash\n git add .\n git commit -m "Fix review comments"\n git push\n`            | Make changes → Commit → Sync                                |
| **9. Merge PR**       | Integrate approved code into main                       | *(GitHub UI → Merge)*                                                            | Done in browser                                             |
| **10. Cleanup**       | Remove unused branch (repo hygiene)                     | `bash\n git checkout main\n git pull\n git branch -d feature/login-validation\n` | Switch to main → delete branch via branch menu              |
